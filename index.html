Getting Started

To run the full Japan Planner application:

  1. Make sure you have Node.js installed
  2. Run npm install to install dependencies
  3. Run npm run dev to start the development server
  4. Open http://localhost:3000 in your browser

Desktop Application

To run the desktop version:

  1. Double-click the JapanPlanner.bat file
  2. The application will open in a desktop window

If you encounter issues, try running the diagnostic tool:

  1. Double-click the Diagnostic.bat file
  2. Check the logs for any error messages
